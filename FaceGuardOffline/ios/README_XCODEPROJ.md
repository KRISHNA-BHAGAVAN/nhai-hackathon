# iOS Xcode Project Setup

The `.xcodeproj` file is auto-generated by React Native's `react-native init` command.

To set up the iOS project:

1. Run `npx react-native init FaceGuardOffline --template react-native-template-typescript --version 0.74.5`
   in a temporary directory
2. Copy the `ios/` folder structure (the `.xcodeproj`, `.xcworkspace`, and native source files)
3. Replace `ios/Podfile` with the one in this repo
4. Run `cd ios && pod install`
5. Open `ios/FaceGuardOffline.xcworkspace` in Xcode
6. Add the `assets/models/*.tflite` and `assets/models/*.task` files to the Xcode project bundle
   (drag into Xcode, check "Copy items if needed", add to target FaceGuardOffline)
7. Build and run

## Metal Framework
Metal is automatically linked on iOS 12+. No additional configuration needed.
The `MTL_FAST_MATH = YES` flag in Podfile's post_install hook optimises Metal shader compilation.

## TFLite GPU Delegate
react-native-fast-tflite uses the Metal delegate on iOS automatically when
the `'gpu'` parameter is passed to `loadTensorflowModel`. This is configured
in FaceDetector.ts, FaceRecogniser.ts, and LivenessDetector.ts.
